**Проект: веб-сервис для интеллектуальной генерации заключения врача-кардиохирурга по данным КТ**

Задача: Разработать веб-сервис, который принимает от пользователя набор изображений (срезы КТ аорты) и таблицу измерений, а затем автоматически формирует структурированный медицинский отчёт в виде PDF и/или HTML по заданному шаблону. Генерация текста отчёта выполняется LLM, качество отчёта оценивается пользователем.

Пользовательский сценарий: 
1) Пользователь загружает кейс: изображения (PNG/JPEG, возможно ZIP) + измерения (CSV/JSON) + базовые метаданные пациента (ФИО/шифр, возраст, пол, дата исследования, анамнез).
2) Сервис валидирует входные данные (структура измерений, единицы, диапазоны значений).
3) Из клинических протоколов подтаскивается дополнительный релевантный контекст 
3) LLM генерирует текстовые блоки отчёта по шаблону: диагноз/описание, таблица ключевых размеров, выводы, рекомендации.
4) Генерируется итоговый отчёт: HTML и PDF в формате, похожем на предоставленный пример.
5) Пользователь оставляет оценку отчёта (например, 1–5 + комментарий по качеству/ошибкам).
6) В базе сохраняются: входные файлы/ссылки, извлечённые измерения, версия промпта/модели, трассировки LLM, итоговый отчёт и оценки.
