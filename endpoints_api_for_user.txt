/interact_llm
description:
Разархивирует данные
Генерирует промпт
Отсылает промт в llm
Получает ответ от llm
Отправляет Ответ от llm и данные трассировки в /diagnosis
Получает id_report от /diagnosis
Отдаёт id_report во фронт

request:
Измерения/zip

response:
200: id_report
400: ne Ok

Записывает фото на диск

отдаёт id_report в /interact_llm

request:
Измерения+фото+метаданные/zip

response:
200: Ok
400: ne Ok
